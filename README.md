# Book Scraper - BeautifulSoup

Scrape book data from books.toscrape.com

## Features
- Scrape 1000 books across 50 pages
- Filter by rating (4-5 stars only)
- Export to Excel (.xlsx)

## Libraries Used
- requests
- beautifulsoup4
- openpyxl

## Output
375 books collected with title, price, and rating
